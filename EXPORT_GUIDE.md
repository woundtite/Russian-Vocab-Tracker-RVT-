## ðŸ”§ Troubleshooting Exports

### File Won't Download
- Check browser security settings
- Try a different browser
- Disable pop-up blockers
- Check Downloads folder

### Anki Won't Import
- Make sure file is `.txt` format
- Check Anki is up to date
- Restart Anki application
- Try copying format: Russian[TAB]English[TAB]Example

### Quizlet Won't Parse
- Make sure you're importing as "From a file"
- Check file is `.txt` format
- Verify tab-separation between terms
- Ensure UTF-8 encoding (should be automatic)

### Data Lost After Export
- Always keep JSON backups
- Don't clear browser data without backup
- Use import to restore from backup
- Export multiple copies to cloud
