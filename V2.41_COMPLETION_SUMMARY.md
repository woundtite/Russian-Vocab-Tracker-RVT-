## Developer Notes

### Key Implementation Insights

1. **Synonym Storage**: JSON stringification in select option values enables complex data in dropdowns

2. **Modal State Management**: Storing currentVerbEntry and currentConjugationTab allows seamless tab switching

3. **Gender in Past Tense**: Keys like "Я (m)" and "Я (f)" maintain clarity while being simple to parse

4. **Backward Compatibility**: Keeping conjugations field ensures old code never breaks

5. **Tab Switching**: CSS grid conjugations reuse same DOM, only data source changes

### Best Practices Applied

- Single Responsibility: Each method has one clear purpose
- DRY Principle: displayConjugationChart() used for both tabs
- State Management: Instance variables track current state
- Separation of Concerns: Data, UI, and logic are separated
- Progressive Enhancement: Works without JavaScript, better with it
