## Key Features Explained

### Feature: Synonym Selector
- **Why**: Helps users understand nuances (beautiful vs handsome)
- **How**: SYNONYMS database + dropdown UI
- **Where**: Form input area (below translation field)
- **Docs**: [V2.41_QUICKSTART.md - Synonym Selector](V2.41_QUICKSTART.md), [V2.41_IMPLEMENTATION.md - updateSynonymDropdown](V2.41_IMPLEMENTATION.md)

### Feature: Verb Aspects
- **Why**: Understand duration and completeness of actions
- **How**: aspect field in RUSSIAN_VERBS + modal badge
- **Where**: Modal header (below verb name)
- **Docs**: [V2.41_RELEASE_NOTES.md - Verb Aspects](V2.41_RELEASE_NOTES.md), [V2.41_IMPLEMENTATION.md - Aspect Field](V2.41_IMPLEMENTATION.md)

### Feature: Verb Directionality
- **Why**: Understand motion verb distinctions (–∏–¥—Ç–∏ vs —Ö–æ–¥–∏—Ç—å)
- **How**: directionality field in RUSSIAN_VERBS + modal badge
- **Where**: Modal header (alongside aspect)
- **Docs**: [V2.41_RELEASE_NOTES.md - Verb Directionality](V2.41_RELEASE_NOTES.md), [V2.41_IMPLEMENTATION.md - Directionality Field](V2.41_IMPLEMENTATION.md)

### Feature: Past-Tense Conjugations
- **Why**: Learn complete verb conjugation system
- **How**: pastTense object with 11 gender-aware forms + tab UI
- **Where**: Modal "Past Tense" tab
- **Docs**: [V2.41_RELEASE_NOTES.md - Past-Tense Conjugations](V2.41_RELEASE_NOTES.md), [V2.41_IMPLEMENTATION.md - Past-Tense](V2.41_IMPLEMENTATION.md)

## Frequently Asked Questions

**Q: Do I need to migrate my v2.4 data?**
A: No! v2.41 is fully backward compatible. Your data loads automatically.

**Q: Why does the synonym dropdown not appear?**
A: The word may not be in the SYNONYMS database yet. Check [V2.41_QUICKSTART.md](V2.41_QUICKSTART.md) for supported words.

**Q: What's the difference between unidirectional and multidirectional?**
A: See [V2.41_RELEASE_NOTES.md - Verb Directionality](V2.41_RELEASE_NOTES.md)

**Q: How do I add more synonyms?**
A: Edit SYNONYMS object in script.js. See [V2.41_IMPLEMENTATION.md - SYNONYMS Object](V2.41_IMPLEMENTATION.md)

**Q: Where's the perfective aspect?**
A: Coming in v2.5+. See [V2.41_RELEASE_NOTES.md - Planned Enhancements](V2.41_RELEASE_NOTES.md)
