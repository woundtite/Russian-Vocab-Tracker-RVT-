## Features

### 1. **Live API Integration**

All data is fetched in real-time from trusted public APIs:

#### OpenRussian API
- Fetches verb conjugations automatically
- Returns present tense (6 forms) and past tense (4 gender-aware forms)
- Includes aspect (perfective/imperfective) and directionality (uni/multidirectional)

#### MyMemory API
- Provides English translations
- Returns alternative translations if available

#### Wiktionary API
- Fetches Russian and English synonyms
- Helps find related words

#### Accent.aot.ru API
- Provides stress mark information
- Automatically bolds the stressed syllable (e.g., –º–æ**—Å–∫–≤–∞**)

### 2. **"Did You Mean?" Spell-Check**

Both Russian and English inputs get intelligent suggestions:
- **Levenshtein Distance Algorithm**: Suggests words with ‚â§2 character edits
- **Russian Suggestions**: Database of 27 common Russian words
- **English Suggestions**: Database of 27 common English words
- Click any suggestion to auto-fill the form

### 3. **Smart Word Categories**

Organize your vocabulary:
- üìã **Review** ‚Äî Words needing practice
- üìö **Supplemental** ‚Äî Extra vocabulary
- ‚≠ê **Critical** ‚Äî High-priority words
- üìñ **Lesson** ‚Äî Current lesson words

### 4. **Conjugation Explorer**

View full verb conjugations in an interactive modal:
- **Present Tense Tab**: —è, —Ç—ã, –æ–Ω/–æ–Ω–∞/–æ–Ω–æ, –º—ã, –≤—ã, –æ–Ω–∏
- **Past Tense Tab**: Gender-aware forms (–º, –∂, –Ω, –º–Ω)
- **Stress Marks**: Bolded for easy pronunciation

### 5. **Vocabulary Management**

- Add words with live data fetching
- Search by Russian word, English translation, or example sentence
- Filter by category (Review, Supplemental, Critical, Lesson)
- View statistics (total words, breakdown by category)
- Expandable synonym sections
- Delete words individually

### 6. **Export & Import**

- **Export JSON**: Download your entire vocabulary as a backup
- **Import JSON**: Restore from backup file
- **Timestamped Exports**: Files include date (e.g., `rvt-vocab-2025-02-03.json`)

### 7. **Data Persistence**

- All vocabulary stored in browser's `localStorage`
- Survives browser restarts
- Easy JSON export for permanent backups
- No cloud account required

## Troubleshooting

### App doesn't open automatically
- **Manual start**: Open `http://localhost:8000` in your browser
- **Port 8000 in use**: Try `./rvt.sh` again or kill the process:
  ```bash
  lsof -ti:8000 | xargs kill -9
  ```

### API requests failing silently
- **Check internet connection**: App requires live web access
- **API down**: Try a different word or refresh the page
- **Console errors**: Open DevTools (F12) ‚Üí Console tab to see errors

### Words not saving
- **Check browser storage**: Some browsers have low limits or private mode restrictions
- **Export first**: If storage is full, export JSON then clear data
- **Try different browser**: Firefox, Chrome, and Safari all work

### Spell-check not showing suggestions
- **Type slower**: Suggestions appear as you type after 2+ characters
- **Check spelling**: Levenshtein distance only suggests words ‚â§2 edits away
- **Review wordlist**: Suggestions limited to ~27 common English/Russian words

## Known Limitations

1. **No offline mode**: Requires internet for live API calls
2. **API availability**: If APIs go down, new words can't be added
3. **No accounts/sync**: Data stored locally only (use JSON export for sync)
4. **Stress marks**: Some words may have missing or incorrect stress data
5. **Conjugations**: Irregular verbs sometimes return incomplete data
6. **Mobile**: UI not optimized for phones (works, but cramped)
