## Troubleshooting

### Synonym dropdown not appearing?
- Check that the word exists in the SYNONYMS database
- Verify word spelling matches (case-insensitive)
- Check browser console for JavaScript errors

### Past-tense forms showing zeros?
- Ensure the verb is in RUSSIAN_VERBS database
- Check that pastTense object is properly formatted
- Verify no syntax errors in script.js

### Aspect/Directionality not showing?
- Modal header should display metadata below verb name
- Check that verb object has aspect and directionality fields
- Verify styles.css includes metadata styling
